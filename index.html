PROJECTS

  • Design and production of an ultra fuel effecient Electric vehicle, and a Hydrogen vehicle.

    The project aims to develop a ultra fuel effecient electric vehicle prototype and then converting that vehicle into a Hydrogen Fuel Cell powered vehicle.

    Project Supervisor: Dr.Dhiraj.K.Mahajan, Associate Professor, IIT Ropar
  • Production of Micro-Pin Fins using reverse EDM, and checking its heat dissipation properties for different shapes of pin-fins.

    The project aims to manufacture Micro-Pin Fins using Reverse EDM process, and check there heat transfer properties for different shapes and sizes of the Pin-Fins.

    Project Supervisor: Dr.Anupam Agarwal, Associate Professor, IIT Ropar
  • Development of Real-Time Railway track crack detection system.

    The project aims to develop a real-time railway track crack management system, which can detect track crack at some distance from the current location of the running train, and would automatically stop the train and also inform the control-station and the driver about the crack ahead.

    Project Supervisor: Dr.Himanshu Paliwal, Assistant Professor, IIT Ropar
  • Calculating the surrounding conditions (like Temperature, Relative Humidity and AQI) in an ICU most optimal for a patient suffering with corona virus.

    The project aims in calculating the best surrounding conditions of an ICU for a patient suffering with Covid-19 virus to help their speedy recovery. The project also aims to help building the most optimal negative pressure by the help of those surrounding conditions.

    Project Supervisor: Dr.Navin Gopinathan, Assistant Professor, IIT Ropar
  • Development of a software to utilize CFD in simulating body fluids.

    This project aims at develop a software to assist the doctor in simulating the flow of different fluids in human body, and to bridge the gap between a Mechanical engineer and a Doctor. This software would assist the doctor in planning an operation or selecting a prosthetic/related machine suitable for a specific person.

    Project Supervisor: Dr.Javed.N.Agrewala, Professor, IIT Ropar
